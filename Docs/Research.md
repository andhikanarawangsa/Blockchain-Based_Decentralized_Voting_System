# Research & Future Work

## Contribution
This project demonstrates a decentralized voting system using blockchain, Proof-of-Work consensus, and cryptographic authentication.

## Research Scope

- Distributed systems
- Blockchain consensus mechanisms
- Cryptographic voting systems
- Peer-to-peer synchronization

## Threat Model

- Sybil attacks
- Double voting
- Blockchain tampering
- Replay attacks
- Network partitioning

## Future Work
- Byzantine Fault Tolerance (BFT)
- Smart contract-based voting
- WebSocket real-time sync
- Docker cluster deployment
- Identity verification layer (KYC simulation)
- Zero-knowledge voting system
